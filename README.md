# Fake News Detection using Machine Learning
This project aims to classify news articles as Fake or Real using machine learning techniques.
It uses text preprocessing, TF-IDF vectorization, and a Logistic Regression model.

## How to Run the Project
1. Make sure Python is installed on your machine.

2. Make sure you are inside the project directory

3. Install the required libraries:
pip install pandas scikit-learn nltk gradio

4. Run the web application:
python app_gradio.py
